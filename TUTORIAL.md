# Scholar's Dungeon 新手指南

欢迎来到 **Scholar's Dungeon**！这是一个将学习过程游戏化的工具，旨在通过 RPG 元素提升你的专注力和学习动力。

## 1. 核心玩法：Explore Mode (探索模式)

这是你最常使用的功能。

*   **专注计时器**：在 Explore 界面，设置你的 Focus (专注) 时间。
*   **休息模式**：如果开启了 Rest，专注时间结束后会自动进入休息倒计时。
*   **Dungeon 进度**：在全屏模式下，你可以看到当前 Dungeon 的进度条。

## 2. 进阶成长：Dungeons & Talents (地下城与天赋)

*   **Dungeons**：在 Dungeon 界面管理你的学习挑战。Major Dungeon 包含多个 Sub Dungeon，完成所有子任务即可通关 Major Dungeon。
*   **Talent Tree (天赋树)**：
    *   **Truth Crown (A)**：提升 XP 获取。
    *   **Golden Law (B)**：提升金币获取。
    *   **Fate Dice (C)**：增加随机性和额外奖励。
    *   使用 Talent Scrolls 解锁天赋，并激活它们以获得增益。

## 3. 经济系统：Merchant's Outpost (商店)

*   **Coins (金币)**：通过完成学习 Session 和 Dungeon 任务获得。
*   **Shop**：购买功能性道具。
*   **Gacha & Ichiban**：消耗金币进行抽奖，获取稀有奖励。

## 4. 数据安全：Astral Archives (云同步)

*   **云同步**：进入 Settings -> Astral Archives，使用你的 Secret Code 绑定云端。
*   **自动检测**：每次打开应用时，系统会自动检测本地与云端存档是否一致。
*   **手动下载**：如果需要，可以在 Astral Archives 界面点击 "Download from Cloud" 强制从云端恢复数据。

## 5. 任务与成就

*   查看 Quest Board 了解每日、每周任务及成就。完成任务可获得 XP、金币或稀有道具。
